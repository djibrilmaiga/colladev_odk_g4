User Story : UI/UX de l’application mobile CollabDevEn tant qu’un developpeur mobile de CollabDev,Je veux une application qui offre une experiance optimisée, avec une interface claire et intuitive.Afin de faciliter la naviguation entre les differante pages de l'application.Definition of done :1.L’application mobile dispose d’une page d’accueil avec les projets en cours et les contributions récentes.2.interface doit etre responsive et sadapter aux differentes tailles d ecran.3.Les couleurs et les polices doivent être cohérentes avec la charte graphique.4.Les boutons doivent être facilement accessibles et lisibles.5.Le parcours utilisateur doit être intuitif avec une navigation fluide.6.Les éléments importants (projets, notifications, profil) doivent être facilement identifiables.
